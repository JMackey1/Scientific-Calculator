Getting Started

Scientific Calculator is a Windows calculator program that does many of the typical scientific calculator functions you would expect: trigonometry functions, exponentiation, roots, and more.



1: History. 2: Current expression. 3: Interface buttons.

While the calculator has buttons for all of the main functions, it is designed for quick and easy keyboard use as well.

You can scroll through the sections to read about all of the functionality of the calculator, or use the navigation panel on the left to jump to a given section.

Interface Buttons

del

The del button will remove a single input character from the current input's cursor position.

CE (clear entry)

The CE button will delete the current expression's contents.

C (clear all)

The C button will delete the current expression's contents as well as the history's contents.

±

The ± button will enter negation( into the current expression.

0-9

The numeric buttons (0-9) will enter the corresponding number into the current expression.

.

The decimal button will enter a decimal into the current expression.

=

The equal button will compute the answer for the current expression. The answer, as well as the expression, will be displayed in the history and the current expression will be cleared.

Operators (/, *, -, +, ^, %, !, √)

The operator buttons will enter the corresponding operator into the current expression.

Note: The √ operator is entered as a fractional exponent.

Left and Right Parentheses

The parentheses buttons will enter a left or right parentheses into the current expression.

π

The π button will enter the π symbol into the current expression.

e

The e button will enter e into the current expression.

Note: This is Euler's number (2.71828).

log

The log button will enter log( into the current expression. This is log base 10.

ln

The ln button will enter ln( into the current expression. This is the natural log (base e).

floor

The floor button will enter floor( into the current expression. This rounds a result down.

ceiling

The ceiling button will enter ceiling( into the current expression. This rounds a result up.

sin

The sin button will enter sin( into the current expression. Trigonometry sine.

cos

The cos button will enter cos( into the current expression. Trigonometry cosine.

tan

The tan button will enter tan( into the current expression. Trigonometry tangent.

asin

The asin button will enter asin( into the current expression. Trigonometry arc sine.

acos

The acos button will enter acos( into the current expression. Trigonometry arc cosine.

atan

The atan button will enter atan( into the current expression. Trigonometry arc tangent.

ans

The ans button will enter ans into the current expression. This represents the previous answer computed.

Keyboard Controls

Not all keys on the keyboard will enter something into the current expression. This is largely to prevent adding junk to the expression that will not make it through the evaluation process.

Instead of clicking on the "sin" button, you can type the keys "s", "i", "n" and "(" to find the sine of the entered expression.

The general idea is that your keyboard will enter into the current expression what you expect it to. If you hold shift and press "1", you will add "!" to the current expression. If you press "*" on the numpad, you will add "*" to the current expression. The notable exception to this is the P key, which enters π, and the arrow keys, which have special behavior.

Below are the keys and what they enter into the current expression.

Letters

Any letter that is part of a function name will be recognized and added into the current expression. See Function Behavior and Examples for a list of functions.

Numbers

Both the normal number keys and the numpad number keys will enter the corresponding number into the current expression.

P

This will enter the π symbol into the current expression.

e

E will enter e into the current expression. Used in some functions, scientific notation and as Euler's number.

Return/Enter

The return and numpad enter keys will try to find the answer to the current expression.

.

The period/decimal key will enter a period/decimal into the current expression.

Numpad Operators (/, *, -, +)

The numpad operator keys will enter the corresponding operator into the current expression.

Shift+Keys

Holding left or right shift and pressing a regular key will enter the alternatvie key symbol into the current expression. Example: Shift+6 will add "^" to the current expression.

Backspace

Backspace will delete a single character from the current expression at its current cursor position.

Space

Space will add a space to the input view. Spaces are ignored by the parser and can be used to make an expression easier to read.

Up Arrow (↑)

The up arrow (↑) will go back through the history and enter the previous result into the current expression. This removes anything in the current expression.

Down Arrow (↓)

The down arrow (↓) will go forward through the history and enter the previous result into the current expression. This removes anything in the current expression.

Left/Right Arrow (←/→)

The left and right arrows (←/→) will move the cursor position in the current expression to the left or right. This can be used to input or remove a character at a specific position.

Ctrl+V

You can copy and paste anything you want to the current expression by using Ctrl+V.

Note: This allows you to enter junk into the current expression.

F1

The F1 key will open the help window (this document).

Current Expression

The current expression area shows the expression that is currently being entered. This is the expression that will be evaluated upon pressing return/enter or the "=" button. The image below shows an example of a complex expression:

The current expression is: sin(cos(3+2))*1.34e+20

The expression is not evaulated in pieces. That is, you can enter a complete expression without it being evaluated after every operator like some calculators do.

You can use the mouse or arrow keys to move the cursor position and enter characters somewhere other than the end of the expression. For example, if you wanted to change the (3+2) term inside the cos function, you could press the left arrow key until the cursor is in front of the "2" and then press backspace to remove 3+2. Finally, you enter the new expression to use with the keyboard or buttons. Alternatively, you could use the mouse and click on the position in front of the "2" to set the cursor there.

When entering characters to the expression in this way, the character will be inserted at the cursor position and the cursor position will be set to after the newly entered character.

For very long expressions, this view becomes scrollable vertically.

History

The history area shows the previously entered expressions and their results (or error messages). Entries are shown with the most recent expression at the top (near the word "History") and older expressions at the bottom. When there are too many entries to fit in the view, the view becomes scrollable vertically.

In this example, the last expression entered was "sin(0)". The first expression entered was "2+2". If you want to use a previous expression for anything, you can navigate through the history using the up/down arrow keys (↑/↓). This can be used to break an expression into parts and see the individual component values, correct an input error or re-evaluate an expression using different terms.

Note: Accessing the history in this way will remove anything currently entered in the current expression. This cannot be used to enter a previous expression into a new expression. If you would like to do this, you can select the item in the history view with the mouse and then use Ctrl+C, Ctrl+V to copy and paste something into the expression.

The history view will contain all expressions previously entered. However, the up/down arrow keys (↑/↓) will only cycle through the last 50 expressions.

Function Behavior and Examples

This section goes over the functions that are available in the calculator, including their syntax and what they do. Functions can be used like operators (i.e. you can multiply by them) and you can create complex expressions using compound functions (such as sin(cos(0))).

Note: Parentheses are not required for functions; sin(5) is the same as sin5. However, it is strongly recommended that you enclose a function's expression with parentheses to make it easier to read.

Roots (square, cube, etc.)

While technically not a function, roots bear special mention. Roots are products of fractional exponents. As such, when you want to find the root of an expression you need to raise that expression to a fractional power. 1/2 is the square root of a number, 1/3 is the cube root of a number, and so on.

Negation

Negation will find the negated value of an expression. In this example, the expression is "-1" and the result is "1".

The syntax for this function is: negation.

Log (Base 10)

Log will find the log (base 10) of an expression. This means "how many powers of 10 do I need to equal the expression". In this example, 10 must be raised to the 3rd power to equal 1000.

The syntax for this function is: log.

Ln (Natural Log)

Ln will find the natural log (base e) value of an expression. This means "how many powers of e (Euler's number) do I need to equal the expression". In this example, e must be raised to the 2nd power to equal e * e.

The syntax for this function is: ln.

Floor

Floor will round an expression down, even for extreme values.

The syntax for this function is: floor.

Ceiling

Ceiling will round an expression up, even for extreme values.

The syntax for this function is: ceiling.

Sine

Sin will find the sine of an expression.

The syntax for this function is: sin.

Cosine

Cos will find the cosine of an expression.

The syntax for this function is: cos.

Tangent

Tan will find the tangent of an expression.

The syntax for this function is: tan.

Arc Sine

Asin will find the arc sine of an expression.

The syntax for this function is: asin.

Arc Cosine

Acos will find the arc cosine of an expression.

The syntax for this function is: acos.

Arc Tangent

Atan will find the arc tangent of an expression.

The syntax for this function is: atan.

Previous Answer

You can use the keyword "ans" to use the previous expression's answer as input to a new expression. In this example, the first expression computed was 2*2*3, which equals 12.

For the second expression, ans was used to find the square of the previous expression. This was done by multiplying ans by itself.

The syntax for using the previous answer is: ans.

Scientific Notation

The calculator allows you to enter a number in scientific notation. Scientific notation is defined as: exactly 1 non-zero integer, a period, any number of integers, the letter E (upper or lower case), a plus or minus sign (+/−) and an integer representing how many powers of ten to multiply the number by.

In this example, we use the "+" symbol to make the number larger (1.2345 * 10^4). By using the "-" symbol, we can make a number smaller. For example, "12345E-4" would evaluate as "12345 * 10^(-4)", or 1.2345.
